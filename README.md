# pwnedmee
Another pwned tool, using https://ghostproject.fr project.

# Install
```
pip install -r requirements.txt 
```

# Use
```
python pwnedmee.py -u <user/email>
python pwnedmee.py --help
```

# Color
  - green: the user
  - yellow: similar users (--warning)
  - red: all others results
